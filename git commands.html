Instruction Page

'ls -h' for showing hidden files.

'git checkout main(branch name)' to switch branches.

'git init' create an empty Git repository or reinitialize an existing one.

'git log' to see commit history.

'git tag -a (tag name eg: v1.0) -m "message"' to create an annotated tag which stores your name, the date, and a message.

'git tag' to see all tags in your repository.

'git push origin (tag name)' to push tags to remote repo, as tags will otherwise exist only locally.

Pushing commits with git push does not push your tags hence we need to push it explicitly.

'git push --tags' pushes all the tags at once.
